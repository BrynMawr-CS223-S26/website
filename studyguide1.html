7) What is the output of this program?

#include <stdio.h>

int arg_modifier(int x, int *y);

int main() {
    int val1, val2, ret;

    printf("Enter a value: ");
    scanf("%d", &val1);
    printf("Enter another value: ");
    scanf("%d", &val2);

    // pass val1 by value and val2 by pointer:
    printf("before call: val1 = %d val2 = %d\n", val1, val2);
    ret = arg_modifier(val1, &val2);
    printf("after call:  val1 = %d val2 = %d ret = %d\n", val1, val2, ret);

    return 0;
}

int arg_modifier(int x, int *y) {
    printf("  in arg_modifier:      x = %d *y = %d\n", x, *y);
    *y = *y + x;
    x = x + 5;
    printf("  leaving arg_modifier: x = %d *y = %d\n", x, *y);

    // Draw function stack here
    return x;
}

  • Implement a program, ask_matrix.c, that asks the user for a number of rows and columns, creates a 2D matrix, and then asks the user for its values.

  • Write code to reverse an array

$ ./reverse_numbers
Enter the size of the array: 4
Enter a number: 2

  • Write code that implements a power function (for positive integer exponents only). A call to your function should compute \(base^{exp}\) and might look like:

result = power(base, exp);

  • Write code that implements a function void removeLetter(char* str, char letter) that removes all occurrences of letter from str. Your function should modify the contents of str. (To check your work, implement your program and test it from main).

  • Write code that implements a function isPalindrome that returns 1 if a given string is a palindrome and 0 otherwise. Your function can be iterative or recursive. Test your function from main.

  • Write a program, countsort.c, that asks the user for a series of digits and then prints the digits in sorted order. Your program should count the number of times each digit appears in the input and then use these counts to print the sorted list.

$ ./countsort
Enter digits: 034567139432
0 1 2 3 3 3 4 4 5 6 7 9
$ ./countsort
Enter digits: 3212
1 2 2 3

Requirements/Hints:

  • You can assume that inputs will be less than 31 characters

  • Your program should not have memory errors and have good style.

  • You cannot assume that inputs will be digits only

  • Use the ASCII values of the input to convert from a character to an integer.

  • Your algorithm should be O(n)

  • Write a program that takes a list of words as command line arguments and outputs the largest one.
